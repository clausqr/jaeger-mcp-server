# Contributing

Thank you for considering contributing to jaeger-mcp-server.

## How to contribute

- **Build, lint, test, and run locally:** See the [Development](README.md#development) section in the README. It covers install, build, lint, test, and how to run the MCP inspector or scripts against a local Jaeger.
- **Pull requests:** See the [Contribution](README.md#contribution) section in the README for the workflow (fork, branch, PR) and CI expectations.

## Questions

Open a [GitHub Issue](https://github.com/clausqr/jaeger-mcp-server/issues) for bugs, feature requests, or support.
